Detailed Description

template<class T>
class Bst< T >

A templated class implementing a Binary Search Tree (BST).

  • This class manages the tree structure, providing methods for insertion, deletion, searching, and different traversal orders. It adheres to the Rule of Three by providing a destructor, copy constructor, and copy assignment operator.
  • Template Parameters
    TThe type of data elements stored in the BST. This type must support comparison operators (<, >, ==).
